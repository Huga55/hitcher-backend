Приложение для нахождения попутчиков в Санкт-Петербурге

Hitcher

- Главная страница, на ней будет кнопка авторизоваться (войти). Возможно политика. Тайтл (Найди попутчика). Здесь вы можете найти попутчиков
по городу Санкт-Петербург абсолютно бесплатно (сервис никаких комиссий не берет). Написать в поддержку.
- Регистрация только по email + ввод логина (буде виден др пользователям) Сделать колонку БАН, для умников.
- Авторизация и регистрация - попапы. + восстановить пароль?
- После авторизации: Создать тикет. Просмотр тикетов(список тикетов). Мои сообщения(сообщения связаны по пользователям). Настройки акк.
- Настройки акк. Логин, изменить логин. Емаил, изменить емаил. Хочу получать уведомления на почту.
- Создать тикет: откуда - куда(дадата). Дата, время(от - до). Радиокнопки, пассажир, водитель. Кнопка создать.
- Просмотр тикетов: список тикетов, список будет добавляться по скроллу. Возможно будет поиск(по адресу). Чекбоксы найти пасажиров
 или вожителей. Карточки с откуда - куда,
	дата, время, логин создателя, в качестве пассажира или водителя. Написать автору в чат.
- Чат: За грубость, нецензурную лексику, спам акк будет заблокриован навсегда. Окно чата, текстареа для воода. 
	Пожаловаться адмнистрации на собеседника.
